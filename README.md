<h2>Система Анализа Списка Ремонтов Банкоматов</h2>

<h3>Описание задачи</h2>
  <p>Реализовать web приложение на Spring</p>

<h4>Требования</h4>
  <ul>
    <li>Приложение хранит в таблице данные о предметной области. В нашем случае – это список ремонтов банкоматов</li>
    <li>Параметры подключения к БД считывались из конфигурационного файла</li>
    <li>В UI есть 2 основные страницы: страница управления и страница отображения данных</li>
    <li>Аналитические функции необходимо реализовать средствами java</li>
    <li>Дополнительно: возможность редактирования уже загруженных данных в UI</li>
  </ul>

<h5>Страница управления</h5>
  <ul>
    <li>Кнопки: «Загрузить», «Отобразить данные», «Удалить данные»</li>
    <li>По кнопке «Загрузить»: выбирается xls файл определённого формата (файл приложен, есть колонки разных типов данных: строки, числа, даты), выполняется его загрузка в БД. По результату выводится количество загруженных записей</li>
    <li>По кнопке «Отобразить данные»: осуществляется переход на страницу отображения данных</li>
    <li>По кнопке «Удалить данные»: удалить все, что было загружено</li>
  </ul>

<h5>Страница отображения данных</h5>
  <ul>
    <li>Кнопки: «Показать все», «Показать ТОП 3 причины», «Показать ТОП 3 времени ремонта», «Показать повторные ремонты»</li>
    <li>По кнопке «Показать все»: вывести все загруженные данные в виде grid-а. Дату при выводе представить в формате дд.мм.гггг чч.мм</li>
    <li>По кнопке «Показать ТОП 3 причины»: вывести сгруппированные по причинам неисправности данные в виде grid-а. В список должны попасть только 3 наиболее часто встречающиеся причины неисправности</li>
    <li>По кнопке «Показать ТОП 3 времени ремонта»: вывести 3 наиболее долгих ремонта</li>
    <li>По кнопке «Показать повторные ремонты»: вывести все ремонты у которых причина поломки повторилась в течение 15 дней</li>
    <li>По умолчанию отображается весь список данных</li>
</ul>

<h3>Технологии:</h3>
  <ul>
    <li>Spring</li>
    <li>Hibernate</li>
    <li>Maven</li>
  </ul>

<h3>Инструкция по сборке и запуску приложения:</h3>

<h4>Docker Compose</h4>
    <ul>
        <li>Выполнить команду в терминале <b>docker-compose up</b> из папки проекта</li>
        <li>Приложение будет доступно на <b>localhost:8080</b></li>
    </ul>

<h4>Локальное развертывание для тестирования и отладки</h4>
<ul>
    <li>установить JDK-21</li>
    <li>установить PostgreSQL
        <ul>
            <li>создать базу "atm"</li>
            <li>создать базу "atm-test" для тестов</li>
        </ul>
    </li>
    
</ul>
